# ft_ls

My implementation of [ft_ls](ft_ls.en.pdf).

### Goals
Recode the command “ls”

## Installation

### Requirements
* GNU make
* GNU C Compiler (GCC)

## Usage
```
git clone https://github.com/mihailgolban/ft_ls.git
cd ft_ls
make
./ft_ls [file ...]
```
